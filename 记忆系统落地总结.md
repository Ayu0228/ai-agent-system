## 怎么用

```python
from src.memory.gateway import get_gateway

gw = get_gateway()

# 任务记忆
task = gw.create_task("tech-dev")
gw.update_task_status(task.task_id, "planning")
gw.mark_task_done(task.task_id)

# 知识记忆
from src.shared.models import KnowledgeQuery, KnowledgeTriple
triple = KnowledgeTriple(subject="Python", predicate="是一种", object="编程语言")
gw.add_knowledge_triple(triple)
results = gw.search_knowledge(KnowledgeQuery(query_text="编程语言"))

# 经验记忆
from src.shared.models import ExperienceCard
card = ExperienceCard(owner_agent_id="tech-dev", scenario="部署", approach="...", result="成功", lesson="先测试")
gw.store_experience(card)
gw.memify()  # 每周清理

# 用户记忆
from src.shared.models import UserFact, MemoryProposal
gw.write_user_fact(UserFact(user_id="ayu", entity="偏好", predicate="喜欢", object="深色模式", confidence=0.9))
gw.submit_user_proposal(MemoryProposal(user_id="ayu", entity="习惯", predicate="每天", object="看日报", source_agent="researcher"))

# 检索路由
plan = gw.route_query("阿禹喜欢什么颜色", agent_id="user_1")

# 健康检查
from src.memory.lifecycle import MemoryLifecycle
lc = MemoryLifecycle(gw)
print(lc.health_check())
```

## 架构对齐

完全遵守飞书文档《AI Agent 记忆系统设计方案（最终修订版）》全部 12 个章节：

- 五层语义：上下文、任务、知识、经验、用户
- 三层存储：L1 内存(<1ms)、L2 SQLite(~10ms)、L3 ChromaDB+图谱(~100ms)
- 三层加载：热记忆常驻、温记忆触发加载、冷记忆按需召回
- 检索路由优先级：User(1.0) > Context(0.9) > Task(0.7) > Experience(0.6) > Knowledge(0.5)
- 幂等设计：session_step_id 唯一索引 + 原子状态更新
- 自进化双轨：self-improving(P1) + AutoScale(P2)
- 跨 Agent 访问矩阵 + ACL 控制
- 多 Agent 协同大纲系统
